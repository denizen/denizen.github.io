Bilgisayarınıza Hugo Yüklemek

  • 12/29/2019
  • Hugo  
  • static site generator  
  • jekyll  
  • hugo  
  • github  

Bir önceki yazımda static site generator’lardan bahsetmiştim. Hugo nedir diye merak ediyorsanız o yazıya göz atabilirsiniz.

Bu yazıda ise Windows bilgisayarınıza Hugo nasıl yüklenir bunu öğreneceğiz.

Öncelikle şuradaki adresi ziyaret edip, sayfanın altında yer alan versiyonlardan işletim sisteminize uygun olanı bilgisayarınıza indirin.

C dizinin altına Hugo adında yeni bir klasör oluşturun. Onun altına da bin adında yeni bir klasör oluşturun. Bilgisayarınıza indirdiğiniz zip dosyasının içerisindeki tüm dosyaları bin klasörünün altına kopyalayın.

Aslında Hugo’yu yükledik fakat konsol üzerinden kullanabilmek için yapacağımız son bir işlem kaldı o da Windows’un PATH dizinine Hugo’yu eklemek. Windows 10 için aşağıdaki adımları takip edin (Windows’u İngilizce kullananlar için parantez içerisinde belirttim).

  1. Masaüstünüzdeki Bu Bilgisayar(My Computer) simgesine sağ tıklayıp Özellikler (Settings) deyin.
  2. Sol tarafta bulunan Gelişmiş sistem ayarları(Advanced System Settings) bölümüne tıklayın.
  3. Açılan pencereden Ortam Değişkenleri( Environment Variables) butonuna tıklayın.
  4. … için kullanıcı değişkenleri(User variables) bölümünden path yazısını bulun ve üzerine çift tıklayın.
  5. Açılan pencereden Yeni butonuna tıklayın.
  6. Açılan text kutusuna C:\Hugo\bin yazın ve klavyeden Enter tuşuna basın.
  7. Açılan tüm pencereleri Tamam butonuna basarak kapatın. İşlem tamamdır.

İşleminizin başarılı olup olmadığını görmek için başlat bölümüne cmd yazıp çıkan uygulamayı çalıştırın. Komut satırına aşağıdaki komutu yazın.

hugo version

Hugo versiyon numaranız ekrana geliyor ise işleminiz başarılı şekilde tamamlanmıştır.
