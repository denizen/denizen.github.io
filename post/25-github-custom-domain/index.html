Github Sitenizi Kendi Domainine Yönlendirin

 Tarih : 10.01.2020  |  1 dakika  |  144 kelime

Bugüne kadar Hugo sitemizi yaptık, Github’da yayınlama başladık fakat henüz sitemize Github’un verdiği subdomain ile ulaşılabiliyor. Kendi domainimiz var ise bunu kullanabiliriz. Bunu da yapmak çok basit.

Öncelikle Github’da Settings bölümüne geliyoruz.

Altta Custom domain bölümüne gelip sahip olduğunuz domain ismini yazın ve Save butonuna tıklayın.

Şimdi domain kullanıcı panelimize gelip name server yönlendirmesi yapmamız gerekiyor. Domaininizi satın aldığınız servisin domain yönetim paneline girin. DNS Zone bölümüne tıklayın ve aşağıda görebileceğiniz gibi 4 adet A yönlendirmelerinizi yapın.

Eğer www adresinizin de yönlenmesini isterseniz cname kaydı da oluşturabilirsiniz. Bunun için www.siteisminiz.com adresiniz githubkullanıcıismi.github.io adresine cname kaydı açmanız yeterli.

Hepsi bu kadar. DNS yönlenmesi çoğu zaman 2-3 saatte hallolsa da normal şartlarda 24 saate kadar sürebilir. Yani işlemi yapar yapmaz sitenizi kendi domaininizde görmeyi beklemeyin. Birkaç saat süre verip arada bir kontrol edin. 24 saat dolduğunda siteniz hala gelmediyse domain aldığınız firmaya ticket açarak yardım isteyebilirsiniz.
